# Personal Portfolio Website

A professional, responsive personal portfolio built with HTML, CSS, and JavaScript.

## Included Sections

- About
- Skills
- Projects
- Contact Form (client-side validation)

## Run Locally

Open `index.html` directly in your browser.

## Deploy on GitHub Pages

1. Create a new GitHub repository and push these files.
2. Go to repository **Settings** > **Pages**.
3. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main** (or `master`) and folder **/ (root)**
4. Save and wait for deployment.
5. Your site will be live at:
   `https://<your-username>.github.io/<repository-name>/`

## Deploy on Netlify

### Option 1: Drag and drop

1. Go to [Netlify Drop](https://app.netlify.com/drop).
2. Drag this project folder into the page.
3. Netlify instantly deploys and gives you a public URL.

### Option 2: Connect GitHub repo

1. Push this project to GitHub.
2. In Netlify, choose **Add new site** > **Import an existing project**.
3. Select your GitHub repo.
4. Build settings for this static site:
   - Build command: *(leave empty)*
   - Publish directory: `/`
5. Deploy site.
